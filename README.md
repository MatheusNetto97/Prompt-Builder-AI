
## Objetivo do Projeto

Desenvolver um assistente especializado de Inteligência Artificial utilizando o Google NotebookLM, capaz de atuar como especialista oficial do produto Prompt Builder AI.

O projeto tem como objetivo demonstrar a aplicação prática dos conceitos de:

* Ferramentas de IA
* Engenharia de Prompt
* Gestão de Produtos
* Alta Performance Profissional com IA

## Produto Desenvolvido

### Prompt Builder AI

O Prompt Builder AI é uma plataforma especializada em Engenharia de Prompt criada para auxiliar usuários na construção, análise e otimização de prompts para Inteligências Artificiais Generativas.

O produto utiliza o Método P.R.O.M.P.T. Universal para transformar solicitações genéricas em prompts profissionais orientados a resultados.

## Funcionalidades

* Construção de prompts profissionais
* Diagnóstico de prompts existentes
* Refinamento de comandos
* Aplicação do Método P.R.O.M.P.T.
* Aprendizagem assistida

## Técnicas de Engenharia de Prompt Aplicadas

### Zero-Shot Prompting

Utilizado para responder perguntas diretamente quando a documentação contém informações suficientes para fornecer a resposta.

### Few-Shot Prompting

Utilizado para demonstrar exemplos práticos de criação e melhoria de prompts antes da entrega da resposta final.

## Personalidade do Assistente

O assistente foi configurado para atuar como:

### Especialista

Autoridade máxima sobre o produto.

### Comunicador

Explica conceitos de forma clara, objetiva e educativa.

### Integridade

Utiliza exclusivamente a documentação oficial como fonte de conhecimento.

### Transparência

Reconhece seus limites e direciona usuários para suporte quando a informação não estiver disponível na documentação.
